# METIS Overview Video

This Remotion project renders a 15-second explainer for `Milky2018/metis`.
The composition shows a complete MoonBit CSR graph example, runs it through
`part_graph_kway`, and visualizes the two-way partition result.

## Commands

Install dependencies:

```bash
npm install
```

Preview in Remotion Studio:

```bash
npm run dev
```

Render a layout-check still:

```bash
npm run still
```

Render the MP4:

```bash
npm run render
```

Generate the README GIF preview after rendering the MP4:

```bash
npm run preview:gif
```

The committed video asset is copied to `../../assets/metis-overview.mp4`.

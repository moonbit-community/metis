# Third-Party Sources

This package vendors C sources from:

- METIS, revision `dfded64f24664caa8809cacf416d378112e8867f`
- GKlib, revision `e2856c2f595b153ca1ce9258c5301dbabc4f39f5`

Both upstream projects are copyright Regents of the University of Minnesota and
licensed under the Apache License, Version 2.0.

The vendored files are generated by `scripts/prepare.py`. The script flattens
the upstream source trees, rewrites local include directives, and generates a
configured METIS header with:

```c
#define IDXTYPEWIDTH 32
#define REALTYPEWIDTH 32
```
